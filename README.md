# Makeup Product Recommendation & Trend Insights (Python)

A Python-based data analytics and recommendation system built using Sephora's beauty product data.

## Features
- Data cleaning & preprocessing
- Exploratory data analysis (price distribution, rating trends, top brands)
- NLP-based product similarity using TF-IDF
- Content-based recommendation engine
- Optional Streamlit dashboard for interactive exploration

## Tech
Python, Pandas, NumPy, Scikit-learn, Seaborn, Streamlit


## How to Run
pip install -r requirements.txt
streamlit run app.py
